# neco-mesh -- 数学的詳細

この文書では neco-mesh のうち、1D サイズ制御、2D 制約付き Delaunay 三角形分割、3D 四面体化、浸漬メッシュの骨格だけを扱う。個別のパラメータや実装上の閾値は [README-ja.md](README-ja.md) と API docs を参照。

## 1D メッシュ

`mesh_line` は、最大辺長 $h_{\max}$ を超えないように線分を分割する。長さ $L$ の線分なら分割数は

$$
n = \left\lceil \frac{L}{h_{\max}} \right\rceil
$$

であり、節点は

$$
\mathbf{x}_i
=
\mathbf{x}_0 + \frac{i}{n} L \hat{\mathbf{d}},
\qquad i = 0,\dots,n
$$

に置かれる。つまり 1D では、幾何学的な問題はほぼ「長さ制約を満たすように区間を切る」ことへ還元される。

`mesh_curve` は NURBS 曲線を直接解析的に離散化するのではなく、曲線サンプル列に対して同じサイズ制御を適用する。

したがって最終精度は、曲線サンプリングと辺長制御の両方に依存する。

## 2D メッシュ

### 制約付き Delaunay 三角形分割

`mesh_polygon` と `mesh_region` の核は制約付き Delaunay 三角形分割である。動機は単純で、同じ領域を三角形で埋めるにしても、極端に細長い三角形が多い分割は後段の数値計算に不利になりやすい。Delaunay 分割は、そのような悪い分割を避けやすい基準として使われる。

通常の Delaunay 三角形分割では、任意の三角形の外接円内に他の点が入らないことを条件にする。幾何的には、近くの点が外接円の内側へ入り込むような分割は、対角線の取り方を変えることでより自然な三角形分割へ改善できることが多い。したがって Delaunay 性は、局所的に無理の少ない三角形分割を選ぶ基準とみなせる。

制約付きの場合は、まず元の境界辺を必ず残す。そのうえで内部では可能な限り Delaunay 性を満たすように分割する。境界辺を優先するため、すべての辺で自由な Delaunay 条件を課すわけではないが、境界忠実性と局所的な形状品質を両立しやすい。

NURBS 領域のメッシュ化では、まず境界曲線をポリラインへ落としてから 2D 問題へ移す。したがってここで得られるメッシュは、解析曲線そのものではなく、そのサンプリング近似に対する三角形分割である。

### 内外判定

`point_in_polygon` は半直線交差法を用いる。点から半直線を伸ばし、境界との交差回数を数えると、

- 奇数回なら内部
- 偶数回なら外部

と判定できる。2D メッシュ化では、候補三角形の重心が領域内にあるかどうかの分類にこの考え方を使う。

## 3D メッシュ

3D 側は、閉じた三角形表面メッシュから品質のよい四面体メッシュを作る流れを取る。これは 2D より難しく、単に Delaunay 四面体分割を作るだけでは境界面が保持されない。

### PLC と Delaunay 四面体化

表面メッシュは、頂点・制約辺・制約面からなる区分線形複体 (PLC) とみなせる。まず表面頂点だけで Delaunay 四面体分割を作り、その後で PLC の辺と面を回復する。

逐次点挿入型 Delaunay では、新しい点 $p$ を含む空洞を作るために `insphere` 判定を使う。正向き四面体 $(a,b,c,d)$ に対して

$$
\operatorname{insphere}(a,b,c,d,p)
$$

の符号を調べることで、点 $p$ がその外接球の内側にあるかを判定できる。外接球の内側に入る四面体を取り除き、空洞境界を新しい四面体で張り直せば、再び Delaunay 性を持つ分割が得られる。

向き判定 `orient3d` と `insphere` の頑健さは 3D メッシュ生成の土台であり、ここが不安定だと正体積の四面体を保てない。

### 境界回復

生の Delaunay 四面体分割には、もとの表面の辺や三角形面がそのまま含まれるとは限らない。そこで

- 辺回復
- 面回復

を順に行い、必要に応じて局所フリップや Steiner 点挿入で PLC をメッシュへ埋め戻す。

ここで重要なのは、Delaunay 性と境界忠実性がしばしば競合することである。3D メッシュ化は「まず Delaunay を作り、次に境界制約を回復し、そのうえで品質を整える」という多段階問題として考えるのが自然である。

### 再細分化と品質改善

境界が回復できても、要素品質はそのままでは十分でないことが多い。そこで追加点挿入や局所改善を行い、扁平な四面体を減らす。

代表的な品質量の 1 つは radius-edge ratio

$$
\rho = \frac{R}{\ell_{\min}}
$$

である。$R$ は外接球半径、$\ell_{\min}$ は最短辺長で、$\rho$ が大きいほど細長い、あるいは扁平な要素であることを意味する。加えて最小・最大二面角も品質指標として重要である。

改善段階では局所フリップ、平滑化、辺操作、追加点挿入を組み合わせる。ここでの目標は理論上の最適性ではなく、

- 正の体積を保つ
- 境界を壊さない
- 極端に悪い要素を減らす

という工学的に有用な品質水準を得ることである。

### 浸漬メッシュ

`generate_immersed_mesh` は境界適合四面体メッシュとは別系統の方法である。閉曲面の外側に構造格子を作り、各直方体セルを一定パターンで四面体に分割し、各要素に「内部をどの程度占めるか」という充填率を与える。

この方法では境界そのものを要素面として一致させない代わりに、生成が単純で頑健になる。内外判定は複数方向のレイ判定を組み合わせて行い、方向依存の誤判定を和らげる。

したがって浸漬メッシュは、

- 厳密な境界適合より頑健さを優先したい場合
- 体積分率を使う離散化が許される場合

に向く。一方で、境界形状を要素境界へ正確に一致させたい用途では、前節の品質四面体化とは目的が異なる。
